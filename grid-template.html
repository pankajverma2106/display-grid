Display : Grid (Parent Properties)

grid-template:

A shorthand for setting grid-template-rows, grid-template-columns, and grid-template-areas in a single declaration.

.grid-container  {
  display: grid;
  grid-template:
  "header header header" 300px
  "sidebar-left main sidebar-right" 100px
  "sidebar-left footer footer" 200px  / 150px auto 150px;
  margin: 50px 0;
}
.grid-item.one {
  grid-area: header;
}
.grid-item.two {
  grid-area: sidebar-left;
}
.grid-item.three {
  grid-area: main;
}
.grid-item.four {
  grid-area: sidebar-right;
}
.grid-item.five {
  grid-area: footer;
}
